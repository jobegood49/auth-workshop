
# Learn auth
## REST API Endpoints

| Endpoint     | Method | Description    |
| ------------ | ------ | -------------- |
| `/`          | GET    | Hello          |
| `/users`     | GET    | List of Users  |
| `/login`     | POST   | Login user     |
| `/register`  | POST   | Register user  |
| `/profile  ` | PUT    | Access profile |
